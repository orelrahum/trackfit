# Trackfit - מעקב תזונה חכם

AI-powered nutrition tracker for Israeli food products.

## Quick Start

```bash
npm run install:all
# Add your Gemini API key via the Settings page (or .env file)
npm run dev
```

Open http://localhost:5173

## Stack
- **Backend**: Node.js + Express + SQLite
- **Frontend**: React + Vite
- **AI**: Google Gemini (multimodal - text, image, voice)
- **Database**: 5,300+ Israeli food products from Fuder
